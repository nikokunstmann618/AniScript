# Examples

Short, single-topic samples plus two longer demos. Run with `node src/aniscript.js examples/<file>.ani`.

| File | What it shows |
|------|----------------|
| `hello.ani` | One print |
| `intro.ani` | String + loop |
| `loops.ani` | Counted loop |
| `ifs.ani` | Nested `geass` |
| `functions.ani` | Composed methods on a `move` |
| `gcd.ani` | Recursive GCD |
| `points.ani` | Several `Point`-like instances |
| `scope.ani` | Scopes and names |
| `any.ani` | One method, mixed values |
| `optionals.ani` | Defaults and a small wrapper `move` |
| `thirteen.ani` | Float and large integer literals |
| `world_building.ani` | `world` / `character` / `move` hierarchy |
| `hunterx2.ani` | Longer script |

Else branches use the keyword `counter`.
